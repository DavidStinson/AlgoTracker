How to Setup

  1. Create a PRIVATE repo on GitHub to store all your practice problems
  2. Add the name of your GitHub Username and the new Private repo to the left and click save
  3. Next, you need to create and save a personal access token (this token will not leave your device)
  4. To do this, go to your github account settings, click on the "Developer settings", and then go to the "Personal accesss tokens" section
  5. Click on "Tokens (classic)", "Generate new token", "Generate new token (classic)"
  6. Create a note for your personal access token (we suggest AlgoTracker)
  7. Select the Expiration Time Period (we suggest shorter expiration dates (7 days) for improved security)
  8. Select the full repo scope and click on "Generate token"
  9. Copy the token in the API Token for Github and Save
  10. Refresh the page to make sure all the settings have been saved
